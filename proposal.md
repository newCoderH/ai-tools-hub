# AI 工具对比/测评网站 Proposal

## 一、项目概述

**项目名称**：AI 工具消费决策平台
**核心定位**：中文 AI 工具消费决策引擎，帮助用户发现、对比、评测 AI 工具
**差异化**：不做大而全的目录，做深度对比和决策支持

## 四、产品策略

### 核心功能

| 功能 | 说明 |
|------|------|
| 工具发现 | 按分类/场景/价格筛选 AI 工具 |
| 深度对比 | 功能/价格/体验多维度对比 |
| 真实评测 | 用真实账号做深度测评 |
| 价格中心 | 横向对比各工具定价（Token价格/订阅） |
| 省钱攻略 | 聚合优惠信息、免费额度、API折扣码 |

### 页面结构

```
/                          # 首页
/tools/[slug]              # 工具详情页
/category/[category]       # 分类页（写作/图像/视频/编程/对话/办公）
/tag/[tag]                # 标签页
/compare/[tool1]-vs-[tool2]  # 对比页
/alternatives/[tool]      # 替代工具页
/blog                      # 博客/测评文章
/pricing                   # 价格对比总览
```

## 五、变现模式

### 优先级排序

| 模式 | 优先级 | 说明 |
|------|--------|------|
| 联盟佣金（AI厂商返佣） | ⭐⭐⭐ | OpenAI 5-20%、Claude、Midjourney 等 |

3. 建立 Newsletter 积累订阅者

## 六、技术架构

### 技术栈

| 层级 | 推荐方案 | 理由 |
|------|---------|------|
| 前端 | Next.js 15 (App Router) | SSG+ISR 混合渲染，SEO 最优 |
| 后端 | Next.js API Routes + Supabase | 零独立后端 |
| 数据库 | PostgreSQL (Neon/Supabase) | 水平扩展，ISR 友好 |
| ORM | Prisma | 类型安全，开发效率高 |
| 搜索 | Algolia（初期免费） | 快速迭代，后期迁移 Meilisearch |
| 缓存 | Vercel KV (Redis) | 边缘缓存，全球低延迟 |
| 部署 | Vercel + Cloudflare CDN | 全球加速 |

### SEO 优化

- JSON-LD 结构化数据（Tool、FAQ、Review）
- 动态 sitemap.xml
- ISR 按需刷新机制
- Open Graph + Twitter Card 支持

### 广告位布局

| 位置 | 尺寸 | 说明 |
|------|------|------|
| 首屏下方 | Leaderboard 728x90 | 内容阅读后展示 |
| 正文每 500 字 | 300x250 矩形 | 段落间自然插入 |
| 文末 | 336x280 大矩形 | 高价值位置 |
| 侧边栏 | Sticky 160x600 | 滚动时持续可见 |
